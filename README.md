# dotfiles

I simply use [stow](https://www.gnu.org/software/stow/) to manage these dotfiles.  
To deploy configurations, run, for example, `stow alacritty`.
([`install.sh`](`install.sh`) is under development so im not sure if it works ><)

BTW here is a [great tutorial](https://gist.github.com/andreibosco/cb8506780d0942a712fc) for stowing dotfiles.
